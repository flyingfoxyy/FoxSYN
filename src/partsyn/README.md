# PartSyn - 基于分区的并行综合优化

## 概述

PartSyn 是一个基于分区的并行综合优化命令，它将网表划分为多个分区，对每个分区并行施加优化，最后将优化结果合并。

## 使用方法

```bash
partsyn [-n num] [-v]
```

### 参数说明

- `-n num`: 指定分区数量（必须 >= 2），默认值为 4
- `-v`: 启用详细输出模式

## 示例

```bash
# 读取网表并转换为 AIG
read_blif design.blif
strash

# 使用默认 4 个分区进行并行优化
partsyn

# 使用 8 个分区并启用详细输出
partsyn -n 8 -v

# 查看优化结果
print_stats
```

## 实现细节

### 当前版本特性

1. **随机分区**: 将网表中的节点随机分配到各个分区
2. **并行优化**: 每个分区在独立线程中进行优化
3. **简单合并**: 选择节点数最少的分区作为最终结果

### 优化流程

对每个分区执行以下优化步骤（默认 3 轮）：
1. Balance - 平衡网络
2. Rewrite - 重写优化
3. Refactor - 重构优化

## 未来改进方向

1. **智能分区**: 使用更好的分区算法（如基于连通性的分区）
2. **增量合并**: 合并时保留各分区的优化结果
3. **自适应优化**: 根据分区特性选择不同的优化策略
4. **负载均衡**: 确保各分区大小相近

## 性能

在测试用例 `usb_phy.blif` 上：
- 原始节点数: 353
- 优化后节点数: 238
- 优化率: ~32.6%
